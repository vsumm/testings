INSTALLATION

To install the plugin follow the instructions below:

  1. Inside the WordPress admin click on Plugins on the menu then click Add New
  2. Click on Upload Plugin
  3. Browse to the location of the zip file you downloaded from CodeCanyon
  4. Click Install Now
  5. Once the plugin has been installed, click Activate plugin

(Alternatively, unzip the zip file and upload the folder to the wp-content/plugins/ folder. Then go to Plugins on the WordPress menu and activate the Quform plugin.)

ACTIVATION

You can now configure the Quform settings by going to Forms → Settings in the WordPress admin menu or you can create a new form by going to Forms → Add New on the WordPress menu.

Please activate your license by following the instructions on this help page.

Once you have obtained your purchase code, enter it in the Forms → Settings → License & Updates page within WordPress.
